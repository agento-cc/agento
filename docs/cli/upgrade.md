# Upgrading Agento

## Overview

An agento deployment has two components to upgrade:
1. **CLI tool** (`agento`) — installed via `uv tool install agento-core`
2. **Docker images** (cron, toolbox) — version controlled by `AGENTO_VERSION` in `docker/.env`

## Upgrade to latest version

```bash
# 1. Upgrade the CLI
uv tool install --upgrade agento-core

# 2. Update image version in your project
cd /path/to/your/project
# Edit docker/.env — change AGENTO_VERSION to the new version
# Example: AGENTO_VERSION=0.3.0

# 3. Pull new images and restart
cd docker && docker compose pull && docker compose up -d
```

The `setup:upgrade` command runs automatically in the cron container entrypoint, applying any pending schema migrations, data patches, and cron job updates.

## Upgrade to a specific version

```bash
# 1. Install specific CLI version
uv tool install agento-core==0.2.5

# 2. Set the matching image version in docker/.env
# AGENTO_VERSION=0.2.5

# 3. Pull and restart
cd docker && docker compose pull && docker compose up -d
```

## How it works

The customer compose file (generated by `agento install`) references Docker images via the `${AGENTO_VERSION}` environment variable:

```yaml
services:
  cron:
    image: ghcr.io/agento-cc/agento-cron:${AGENTO_VERSION:-latest}
  toolbox:
    image: ghcr.io/agento-cc/agento-toolbox:${AGENTO_VERSION:-latest}
```

This variable is set in `docker/.env` at install time and can be changed for upgrades. Docker Compose reads `.env` automatically.

## Checking current version

```bash
# CLI version
agento --version

# Running container images
cd docker && docker compose ps --format json | python3 -c "import sys,json; [print(s.get('Image','')) for s in json.loads(sys.stdin.read())]"

# Version in .env
grep AGENTO_VERSION docker/.env
```

## After upgrading

- `setup:upgrade` runs automatically on container start
- Check logs for migration output: `agento logs cron`
- If issues occur, you can rollback by reverting `AGENTO_VERSION` in `.env` and re-running `docker compose pull && docker compose up -d`
